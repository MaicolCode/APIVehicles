API Vehicles Documentation

Welcome to the API Vehicles documentation. This guide will help you understand how to interact with the API.

Base URL

The base URL for all API requests is:

http://localhost:6675/vehicles

Response:

    "message": "Vehicle deleted successfully."
}

Error Handling

The API returns standard HTTP status codes to indicate the success or failure of a request. Common status codes include:

  • 200 OK: The request was successful.
  • 400 Bad Request: The request was invalid or cannot be served.
  • 401 Unauthorized: Authentication failed or API key is missing.
  • 404 Not Found: The requested resource could not be found.
  • 500 Internal Server Error: An error occurred on the server.

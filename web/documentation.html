API Vehicles Documentation

Welcome to the API Vehicles documentation. This guide will help you understand how to interact with the API.

Base URL

https://api.vehicles.com/v1

Authentication

The API uses API keys for authentication. Include your API key in the Authorization header as follows: 

    Authorization: Bearer YOUR_API_KEY

Endpoints

1. Get All Vehicles

Retrieve a list of all vehicles.

    GET /vehicles

Response:

    [
        {
            "id": 1,
            "make": "Toyota",
            "model": "Corolla",
            "year": 2020
        },
        {
            "id": 2,
            "make": "Honda",
            "model": "Civic",
            "year": 2019
        }
    ]

2. Get Vehicle by ID

Retrieve details of a specific vehicle by its ID.

    GET /vehicles/{id}

Response:

    {
        "id": 1,
        "make": "Toyota",
        "model": "Corolla",
        "year": 2020
    }

3. Add a New Vehicle

Create a new vehicle entry.

    POST /vehicles

Request Body:

    {
        "make": "Ford",
        "model": "Focus",
        "year": 2021
    }

Response:

    {
        "id": 3,
        "make": "Ford",
        "model": "Focus",
        "year": 2021
    }

4. Update a Vehicle

Update details of an existing vehicle.

    PUT /vehicles/{id}

Request Body:

    {
        "make": "Ford",
        "model": "Focus",
        "year": 2022
    }

Response:

    {
        "id": 3,
        "make": "Ford",
        "model": "Focus",
        "year": 2022
    }

5. Delete a Vehicle

Delete a vehicle by its ID.

    DELETE /vehicles/{id}

Response:

    {
        "message": "Vehicle deleted successfully."
    }

Error Handling

The API returns standard HTTP status codes to indicate the success or failure of a request. Common status codes include:

  • 200 OK: The request was successful.
  • 400 Bad Request: The request was invalid or cannot be served.
  • 401 Unauthorized: Authentication failed or API key is missing.
  • 404 Not Found: The requested resource could not be found.
  • 500 Internal Server Error: An error occurred on the server.
